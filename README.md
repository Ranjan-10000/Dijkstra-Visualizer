# Dijkstra's Algorithm Visualizer (Pygame)

This is a simple **Dijkstraâ€™s Algorithm Visualizer** built using **Python (Pygame)**. It allows users to set a start node, an end node, and obstacles, then visualize the shortest path calculation.

## How to Run
1. Install dependencies:
   ```bash
   pip install pygame
   ```
2. Run the script:
   ```bash
   python dijkstra_visualizer.py
   ```

## Features
- Click to set the **start node**, **end node**, and **barriers**.
- Press **Space** to start visualization.
- Press **C** to reset the grid.
